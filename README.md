# 数学雷电 - Phaser版

基于Phaser.js框架开发的数学教育射击游戏，通过游戏化的方式提升数学技能。

## 项目特色

- 🚀 激烈的射击战斗体验
- 🧮 通过解题获得强力道具
- ⚡ 多种武器和技能系统
- 🎯 主动选择做题时机
- 📱 支持桌面和移动设备

## 技术栈

- **Phaser 3.70+**: 游戏引擎
- **ES6 模块化**: 现代JavaScript语法
- **Vite**: 快速构建工具
- **Arcade Physics**: 物理引擎

## 项目结构

```
math_raiden/
├── src/
│   ├── main.js              # 游戏入口
│   ├── scenes/              # 游戏场景
│   │   ├── BootScene.js     # 启动场景
│   │   ├── PreloadScene.js  # 预加载场景
│   │   ├── MenuScene.js     # 主菜单场景
│   │   ├── GameScene.js     # 游戏场景
│   │   ├── GameOverScene.js # 游戏结束场景
│   │   └── MathQuestionScene.js # 数学题场景
│   ├── objects/             # 游戏对象
│   ├── systems/             # 游戏系统
│   ├── ui/                  # UI组件
│   └── utils/               # 工具类
│       └── Constants.js     # 游戏常量
├── assets/                  # 游戏资源
├── index.html              # 游戏入口页面
├── vite.config.js          # Vite配置
└── package.json            # 项目配置
```

## 游戏玩法

1. **移动控制**: 使用方向键控制飞机移动
2. **自动射击**: 飞机会自动发射子弹
3. **拾取道具**: 触碰道具会触发数学题
4. **解答题目**: 答对获得强化奖励，答错获得基础奖励
5. **难度选择**: 支持3个数学难度等级
   - G1: 10以内加减法
   - G2: 20以内加减法  
   - G3: 乘法运算
