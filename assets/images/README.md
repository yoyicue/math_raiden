# 战机贴图资源说明

## 目录结构
将战机贴图文件放置在 `assets/images/` 目录下，系统会自动检测并使用这些贴图。

### 📐 当前文件分析

#### 已提供的战机贴图（统一64x64px）
所有战机贴图都采用了统一的64x64像素尺寸，这简化了管理和显示逻辑：

- **优点**: 
  - 统一尺寸便于管理和缩放
  - 足够的细节空间
  - GPU友好的2的幂次方尺寸
  - 文件大小合理（5-6KB）

- **当前实现**: 
  - 所有战机使用相同的基础尺寸
  - 通过代码中的`setScale()`来实现大小差异
  - 坦克敌机在代码中设置为1.2倍缩放

#### 制作规范（基于现有文件）
- **朝向**: 玩家战机向上，敌机向下
- **中心对齐**: 确保战机在64x64画布中心
- **透明背景**: 使用Alpha通道，背景完全透明
- **8位色彩映射**: 当前文件使用8-bit colormap格式
- **文件大小**: 控制在5-6KB范围内

## 回退机制
如果某个贴图文件不存在，游戏会自动使用程序生成的几何图形作为备用：
- 玩家战机：绿色三角形 (40x40px)
- 敌机：不同颜色的方块 (40x40px)
- 子弹：黄色矩形（玩家4x10px）/ 紫色圆形（敌机8x8px）
- 导弹：绿色菱形 (8x16px)

### 🔍 常见问题排查
1. **贴图不显示**: 检查文件名是否正确
2. **显示模糊**: 确认尺寸是否为2的幂次方（64x64px已符合）
3. **背景不透明**: 检查PNG是否包含Alpha通道
4. **颜色偏差**: 确认色彩模式为RGB或colormap
5. **文件过大**: 使用PNG压缩工具优化（当前已优化）
6. **加载缓慢**: 减小文件大小或优化压缩
